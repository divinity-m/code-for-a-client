# GUNNERS ROBOTICS RUN LOGS #
Enter in the time the run was completed in seconds in the input indicating such, then do the same for the number of obstacles hit.
Click "LOG RUN" once the values have been entered to log the run.
If any mistakes were made when logging the run, it can be deleted by hovering over the run, then clicking the delete button that appears.

The best scores and overall stats menu update automatically when a run is added or deleted
