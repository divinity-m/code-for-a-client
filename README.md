# GUNNERS ROBOTICS RUN LOGS #
Enter in the time the run was completed in seconds in the input indicating such, then do the same for the number of obstacles hit.
Click "LOG RUN" once the values have been entered to log the run. If any mistakes were made when logging the run, it can be deleted by hovering over the run, then clicking the delete button that appears.
Click "_____" to compare the stats of all the logged runs and show the best ones.
